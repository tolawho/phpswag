# Security Policy

## Supported Versions

Please check the table below to see which versions of `phpswag` are currently receiving security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within `phpswag`, please DO NOT open a public issue.

Instead, please send an email to the repository owner or maintainers directly. (Please update with your actual email address).

We will try our best to acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible. Once the vulnerability is patched, we will release a security update and acknowledge your contribution.
